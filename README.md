# Creaciones SRO - Proyecto responsive (v2)

Proyecto base en **Vue 3 + TypeScript** con **TailwindCSS**, **Naive UI** y **Netlify CMS** (configuración básica).

## Instalación

```bash
npm install
npm run dev
```

## Desarrollo

- App en http://localhost:5173
- Reemplaza las imágenes en `public/uploads/` por tus fotos reales.
- Subir repo a GitHub y configurar Netlify (Identity + Git Gateway) para usar Netlify CMS.
- Formulario preparado para Netlify Forms: al desplegar en Netlify las sumisiones irán al panel de Netlify.

## Notas

- Diseño mobile-first y totalmente responsive.
- Paleta de colores basada en tonos rosados pastel.
